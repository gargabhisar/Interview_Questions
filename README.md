# Interview Questions

Mobile-friendly .NET / C# interview preparation guide (static HTML).

## Files

- `Interview_Questions.html` — main app
- `index.html` — entry redirect
- `web.config` — IIS default document

## Local use

Open `Interview_Questions.html` in a browser, or serve the folder with any static file host.

## Deploy

Upload all files in this folder to your web host root (e.g. IIS / runasp.net).
